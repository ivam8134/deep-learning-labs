## Overview

This repository contains my work for **Lab 2 of the Deep Learning course**.
The goal of this lab is to better understand how neural networks work by **implementing and experimenting with them directly**.

Instead of only studying the theory, this lab focuses on practical experimentation: building models, modifying parameters, observing how training behaves, and understanding the challenges that appear during training.

Working through these exercises helped me develop a deeper intuition about how neural networks actually learn.

---

## Objectives of the Lab

Through this lab, I aim to:

* Understand the structure of deep neural networks
* Implement neural network models
* Experiment with different model configurations
* Observe how hyperparameters affect training
* Analyze model performance and behavior

The emphasis is on **learning through experimentation** rather than simply running provided code.

---

## Approach

While working on this lab, I try to follow a practical learning approach:

* Writing the code manually instead of copying it
* Modifying parameters such as:

  * number of layers
  * activation functions
  * learning rate
  * number of neurons
* Observing how these changes affect training results
* Debugging errors and improving the model

This process helps build a deeper understanding of how deep learning systems behave in practice.
